## 🚀 Projects

### ⚖️ RAG-Enhanced Legal Assistant for India
> *2026 · Python · LangChain · FAISS · Hugging Face · RAG · LLM APIs · Streamlit · Git*

An AI-powered legal Q&A system delivering accessible, explainable legal guidance to Indian citizens across regional languages — covering IPC, CrPC, and RTI corpora.

- Fused **FAISS vector search** with LLM-driven generation for contextually precise legal provisions
- Built explainability layers that distil dense legal language into plain-language summaries
- Reduced hallucinations through **prompt chaining, iterative query reformulation & optimized chunking**
- Production-grade, enterprise-ready RAG pipeline design
